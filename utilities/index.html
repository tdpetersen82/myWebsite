Why these tools

Quality over quantity

They run on your device

Every tool is plain JavaScript in the page. Whatever you paste — JSON, text, numbers — stays in your browser. There's no upload, no account, and nothing logged.

  • No sign-up required
  • No data sent to a server
  • Works offline once loaded

They're built, not bulk-generated

This is a deliberately small set. Each tool does one job well, with real edge-case handling and a clean interface — rather than a hundred half-finished pages.

  • Exact conversion factors verified
  • Clear errors with line numbers
  • Live results as you type

They fit the rest of the site

Same look, same speed, same no-nonsense approach as the games. Load a page and it's ready — no spinners, no cookie walls, no waiting on a third party.

  • Instant load static files
  • Mobile friendly responsive
  • More tools coming
Everything here runs entirely in your browser. No file you open or text you paste is uploaded or stored. Clearing your browser data won't affect these tools — there's nothing saved server-side.
